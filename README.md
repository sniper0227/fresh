#项目介绍
本项目基于当前流行技术组合的前后端分离生鲜2B商城系统： SpringBoot2+MybatisPlus+SpringSecurity+jwt+redis+Vue的前后端分离的生鲜商城系统， 包含客户管理、商品管理、套餐列表、订单管理、采购管理、库房管理、配送管理、促销管理、基础管理、系统管理、配置管理、统计报表、运营管理、
运营简报等功能，更适合企业或个人二次开发；
